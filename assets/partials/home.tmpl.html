Welcome to guggmeta! This is one of the projects created with the Data API published by Guggenheim Helsinki Design Competition. The main goal of guggmeta is to expose the metadata left behind the PDF documents that were included in the 1,715 anonymous submissions.

Browse the submissions received or conduct your on searches!

This project is open source, which means the code is publicly available, can be reviewed, and anyone can join the project and suggest changes. Check out the repository.
